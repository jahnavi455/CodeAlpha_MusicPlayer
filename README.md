# Music Player Application

## Overview

This project is a Music Player Application developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

The application provides an interactive and responsive music listening experience with features such as play/pause controls, next/previous navigation, playlist management, progress tracking, volume control, and autoplay functionality.

---

## Features

* Play and Pause music
* Next and Previous song controls
* Dynamic song title and artist display
* Individual album cover for each song
* Progress bar with seek functionality
* Current time and total duration display
* Volume control option
* Separate playlist panel
* Playlist song selection
* Autoplay ON/OFF toggle
* Responsive design
* Hover effects and smooth transitions
* Modern user interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Add audio files inside the `songs` folder.
4. Add album cover images inside the `images` folder.
5. Open `index.html` in any web browser.
6. Start listening to music.

## Functionalities

### Play / Pause

Users can start or pause music playback using the play button.

### Next / Previous

Navigate between songs using:

* Next button
* Previous button

### Playlist

A dedicated playlist option allows users to:

* View available songs
* Select and play any song directly

### Album Covers

Each song has its own album artwork that updates automatically when the song changes.

### Progress Bar

The progress bar displays playback progress and allows users to:

* Track song progress
* Seek to any position in the song

### Duration Tracking

Displays:

* Current playback time
* Total song duration

### Volume Control

Users can:

* Open volume settings
* Increase or decrease volume
* Mute music if desired

### Autoplay

Autoplay automatically plays the next song when the current song ends.

Users can:

* Turn Autoplay ON
* Turn Autoplay OFF
